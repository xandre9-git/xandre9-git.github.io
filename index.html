  • Creating a text field for users to input text requires the input element and the type attribute. Example: <input type="text">. Input elements are self closing.

  • Placeholder text is what is displayed before the user has entered anything into the text field. Example: <input type="text" placeholder="First Name">

  • Creating web forms is done with the form element and the action. They are used to submit data to a server. Example: <form action="/url-where-data-is-submitted"></form>

  • Adding a submit button to a form is done with the <button> element and type attribute. Place button element within the form element desired. Example: <button type="Submit">Submit</button>

  • Forms can be required to be filled out before submittal with the required attribute. Example: <input type="text" required>

  • Radio buttons can be used when you only want one response from a selection of responses. They are a type of input which can be nested in <label> elements. This causes the radio button to be associated with the label. The related radio buttons should have the same name attribute so that they are grouped together. Radio groups allow for one response to be selected and the other to be deselected.

    Example:
    <label for="indoor">
    <input type="radio" name="indoor-outdoor">Indoors
    <input type="radio" name="indoor-outdoor">Outdoors
    </label>

    Do you prefer to do exercises indoors or outdoors?

  • Creating checkboxes can be accomplished by the input type 'checkboxes'. They are used for responses that require more than one answer. Checkboxes can be nested in their own labels. This will associate the checkbox element with the respective label. Related checkbox elements should have the same name attribute. It is good practice to put a for attribute on the label to match the id attribute of the associated input element.

    Example:
    <label for="situps">
    <input id="situps" name="likes" type="checkbox">Situps
    </label>

    Which of these exercises do you like?

  • All html pages tell browsers which version of html your page is using. This is done by declaring the DOCTYPE of a page with the <!-- DOCTYPE ...> tag where ... is the version of html. After declaration of the doctype any remaining html code needs to be wrapped in html tags: <html></html>

    • Example:

    <html>

It also includes all head elements: <title>, <style>, <meta>, <link>, <script>, and <base>. The markup relating to content of the page goes within the <body> element.

    • Example:

        </head>

        </body>

CSS

This section displays all the CSS code I have been taught. CSS code can be applied in three ways. First directly to HTML elements with the style attribute, second by placing CSS rules within style tags inside the HTML document, and lastly by creating a separate external style sheet containing all the CSS rules, which is referenced by the HTML document.

Changing Color of Text
Text color can be done by changing the style. I have changed the CSS header above using:

<h2 style="color: dodgerblue">

Using CSS Selectors to Style Elements
A style tag can be placed at the start of an HTML document to apply CSS to whichever element, class, or id you specify. Below is an example of the CSS selector within the style tags modifying the body element of this page.

<style>
 body {
  background-color: black;
  color: lightgray;
}
</style>

Using a CSS Class to Style an Element
A class can be specified in a style tag by the use of a a dot "." and its name. i.e. ".green-text". This class can be applied within the element without the dot, like so:

<h1 class="green-text">Elements</h1>

Classes Allow Styling of Multiple Elements
Once a class is defined, all HTML elements within the document can use the class as seen above.

<style>
 .purple-text {
   color: purple;
}
</style>

Changing Font Size
Font size can be changed as so:

element {
 font-size: 40px;
}

Changing Font Family
The font family can be changed as so:

element {
 font-family: calibri;
}

Importing Non Standard Web Fonts
Custom web fonts can be imported into an HTML page by inserting: <link href="https://fonts.googleapis.com/css?family=Lobster" rel="stylesheet" type="text/css"> before the style tags at the beginning of the HTML document. Afterwards, the unique font can be used as so:

.lobster {
 font-family: Lobster;
}

Specifying How Fonts Should Degrade
Font family can be set in a way that if one font is not available another is used.

"font-family: FAMILY_NAME, GENERIC_NAME;""

Sizing Your Images
Image sizes can be adjusted through CSS. Width is one of the specifications which can be modified.


 .larger-image {
  width: 500px;
 }

Adding Borders Around Elements
Borders can be added to your elements.


.thick-hotpink-border {
  border-width: 5px;
  border-color:palevioletred;
  border-style: solid;
}

Adding Rounded Corners Around Elements
The CSS property 'border-radius' can be used to round corners. It can be specified by pixels (px) or by a percentage.

.round-corner {
  border-radius: 10px;
  border-width: 5px;
  border-color:yellow;
  border-style: solid;
}

Specifying Background Color to a div Element

You can specify an element's background color using the background-color property.

.orangebg {
  background-color:orangered;
  width: 38%;
}

Set the id of an Element
Elements can use id attribute. Include '#idname' in CSS styles section. Id's are used to identify one element, where class can be used to identify more than one.The id attribute has a higher specificity than class and will override class if both are applied to an element with conflicting styles.

#idname {
}

<h1 id="idname">

Adjust the Padding of an Element
All HTML elements are little rectangles in essence. The space surrounding each HTML element can be modified by three important properties: padding, margin, and border.

Padding is used to determine the amount of space between the content and its border.

PRACTICE

#pad {
  padding: 10px;
  border-style: solid;
  border-width: 5px;
  border-color: teal;
  text-align: center;
}

Margin is used to determine the amount of space between an element's border and its surrounding elements.

PRACTICE

#mar {
  padding: 20px;
  margin: 50px;
  border-style: solid;
  border-width: 5px;
  border-color: teal;
  text-align: center;
}

Adding Negative Elements to Margins
If a margin is set to a negative value, the element it is applied to grows in size.

PRACTICE

#neg {
  padding: 20px;
  margin: -10px;
  border-style: solid;
  border-width: 5px;
  border-color: teal;
  text-align: center;
}

Adjusting Padding and Margins for Each Side of Elements
Padding for elements can be adjusted for each side using padding-top, padding-right, padding-bottom, and padding-left. The same can be done for margins using: margin-top, margin-right, margin-bottom, and margin-left.

PRACTICE

#padtrbl {
  padding-top: 1px;
  padding-right: 10px;
  padding-bottom: 15px;
  padding-left: 25px;
  border-style: solid;
  border-color:lightblue;
  margin-right: 1000px;
}

Adjusting Padding and Margins with Clockwise Notation

Using Attribute Selectors to Style Elements
Attributes such as radio and checkboxes can be used as selectors in CSS styles to adjust all corresponding elements. This is done by using the '[attr=value]' selector.

[type='radio'] {
  margin: 10px;
}
