Elements

This section will display all the elements I have learned in HTML and HTML 5
  • Tags are closed with </>
  • The Elements heading was made with a <h1> tag. There are a total of 6 heading tags, <h1> through <h6>
  • Paragraphs are made using <p> tags.
  • Commenting HTML is done by inserting <!-- text to developer which wont be seen by web page user -->
  • HTML5 has more descriptive tags; <main>, <header>, <footer>, <nav>, <video>, <article>, <section> and more
  • <main> tag is used to help search engines find the main content of the webpage
  • Images are inserted into websites by using an <img> element which points to a src (source) attribute.
  • To link an image in the webpage (such as the one seen below) you would need to input "<img src="https://www.example.com/image.jpg" alt="something descriptive."> into the HTML. They are self closing tags. The alt attribute is used for screen readers to improve accessibility and to show if the image fails to load.

  • The <a> (anchor) element is used to link outside content to the page. They use a destination web address call an href attribute. <a href="https://www.google.com"> Google</a>.

    • Google

  • Nesting an anchor element within a paragraph is easy. Find the section you would like to insert a link by inserting <a href="https://www.google.com"> click here </a> for users to click. If you would like for the link to open in a new tab, change the code to have an (target="_blank") attribute before the href attribute like so: <a target="_blank" href="www.google.com">click here</a>.

    • Same Page Example: You can find a free code camp using this link.
    New Tab Example: You can find a free code camp using this link.


  • Anchor <a> elements can also be used to jump to different sections within the webpage. You must assign a href attribute to a hashtag (#) plus the id attribute you want to link to. The id attribute must also be set within the element you wish linked.

    • Example: Elements

  • You can make a Dead Link within an <a> element by using a '#' symbol. This is useful for preparation when you do not have the link available you would like to reference: <a href="#">
    Dead Link

  • Images can be turned into links by nesting the image within an <a> element by typing <a href="www.wherever.com/"><img src="something.com/yeah.jpg" alt="how to make images become links"></a>
  • Unordered lists, or bullet style lists, can be created using the <ul> unordered list element. Each bullet can be created using the <li> tag. This bullet was made using the <li> tag.
  • Ordered lists are created using the <ol> tags. <li> tags are used to list items numerically within the <ol> tags.

  1. One
  2. Two
  3. Three
  • Creating a text field for users to input text requires the input element and the type attribute. Example: <input type="text">. Input elements are self closing.

  • Placeholder text is what is displayed before the user has entered anything into the text field. Example: <input type="text" placeholder="First Name">

  • Creating web forms is done with the form element and the action. They are used to submit data to a server. Example: <form action="/url-where-data-is-submitted"></form>

  • Adding a submit button to a form is done with the <button> element and type attribute. Place button element within the form element desired. Example: <button type="Submit">Submit</button>

  • Forms can be required to be filled out before submittal with the required attribute. Example: <input type="text" required>

  • Radio buttons can be used when you only want one response from a selection of responses. They are a type of input which can be nested in <label> elements. This causes the radio button to be associated with the label. The related radio buttons should have the same name attribute so that they are grouped together. Radio groups allow for one response to be selected and the other to be deselected.

    Example:
    <label for="indoor">
    <input type="radio" name="indoor-outdoor">Indoors
    <input type="radio" name="indoor-outdoor">Outdoors
    </label>

    Do you prefer to do exercises indoors or outdoors?

  • Creating checkboxes can be accomplished by the input type 'checkboxes'. They are used for responses that require more than one answer. Checkboxes can be nested in their own labels. This will associate the checkbox element with the respective label. Related checkbox elements should have the same name attribute. It is good practice to put a for attribute on the label to match the id attribute of the associated input element.

    Example:
    <label for="situps">
    <input id="situps" name="likes" type="checkbox">Situps
    </label>

    Which of these exercises do you like?


  • Radio and checkboxes inputs report their values from the value attribute.

    Example:
    <label for="indoor">
    <input id="indoor" value="indoor" type="radio" name="indoor-outdoor">Indoor
    </label>

    Example:

  • The <div> element is a general purpose container element for other elements. It is closed with a </div> tag.

    Example:
    <div>
    <label for="indoor">
    <input type="radio" name="indoor-outdoor">Indoors
    <input type="radio" name="indoor-outdoor">Outdoors
    </label>
    </div>

  • All html pages tell browsers which version of html your page is using. This is done by declaring the DOCTYPE of a page with the <!-- DOCTYPE ...> tag where ... is the version of html. After declaration of the doctype any remaining html code needs to be wrapped in html tags: <html></html>

    • Example:
    <!-- DOCTYPE>
    <html>
    </html>

  • Defining the Head and Body of an html document is done within the html element through <head> and <body>. The head element includes any markup information about the page. It also includes all head elements: <title>, <style>, <meta>, <link>, <script>, and <base>. The markup relating to content of the page goes within the <body> element.

    • Example:
    <!-- DOCTYPE html>
    <html>
        <head>
            <!-- metadata elements -->
        </head>
        <body>
            <!-- page contents -->
        </body>
    </html>
